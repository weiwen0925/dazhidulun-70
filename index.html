編 者 前 言

《大智度論》卷七十，承接卷六十九〈釋佛母品〉而下，集結四品經論，由「般若為諸佛母」這一命題出發，鋪陳出一條從現象論 → 認識論 → 本體論 → 體用論的完整中觀辯證軸線。

第一品〈釋佛母品第四十八之餘〉把焦點放在「般若如實知眾生」，以經文「出、沒、屈、申」四字總綱，從三界流轉的現象動態，深入到邪見拓撲與十四無記的病理結構，最後收束於「五眾如」的實相層；第二品〈釋問相品第四十九〉由舍利弗的提問切入，「相不能破相、相不能知相」雙重解構般若認識方式，揭示「法界常住」之發現論而非發明論。

第三品〈釋佛母品〉則回頭以原典七節 + 釋論八節 + 中觀邏輯八節的三軌結構，正面論證「般若為諸佛之母」的本體論成立基礎；第四品〈大事起品〉以龍樹辯證階梯（事相 → 譬喻 → 實相）為總持，把前三品的命題收束在「大事即般若體用不二」的中觀體用觀上，完成卷七十的整體閉環。

卷七十 · 第一品
釋佛母品第四十八之餘
般若如實知眾生 · 從現象到實相七層解構
本品要旨

本品承接卷六十九〈釋佛母品〉，以「佛因般若如實知眾生」為核心。經文「出、沒、屈、申」四字總綱，由現象層的三界流轉、進到病理層的邪見拓撲、再到實相層的五眾如，層層揭示般若波羅蜜何以為「諸佛之母」。七章從經文總綱、現象動態、邪見破斥、世間有邊無邊、十四無記、到五眾如與般若母位，建立一條完整的「現象 → 病理 → 實相」垂直透視軸。

承先啟後

前章從「實相層的五眾如」收束在「般若波羅蜜為諸佛母」的母位義；下一品〈釋問相品第四十九〉將承此母位，由舍利弗發起對般若深相的提問，把焦點從「般若是甚麼」轉為「般若應如何被認識」，進入認識論層次的中觀辯證。

卷七十 · 第二品
釋問相品第四十九
提問的哲學密碼 · 般若空慧實相論
本品要旨

承上品「般若為諸佛母」之命題，本品由舍利弗以「既已說空，何須再問」起問，展開對般若深相的雙層辯證：諸天散華致問深般若相、論主以三解脫門（空、無相、無作）為般若入門，進而透過「相不能破相」「相不能知相」徹底解構「相」的認識論結構。末以「法界常住」結歸：般若不是發明，乃是發現；究竟心法在於「不取不捨、通達無礙」。

承先啟後

前章以「不取不捨」收束問相；下一品〈釋佛母品 第四十八〉回頭重新審視「般若佛母」的根源命題：為何般若能為諸佛之母？「知作人」的無作相如何成立？須菩提之難「空何以能生能示」如何破斥？此品以原典 7 節 + 釋論 8 節 + 中觀邏輯 8 節三軌綜攝，把問相的認識論收歸到般若母位的本體論。

卷七十 · 第三品
釋佛母品 第四十八
般若為諸佛母 · 三軌綜攝
本品要旨

本品以「般若為諸佛母 · 諸佛依止」開卷，正面建立般若母位義。經典原文七節梳理「諸佛依止 → 知作人 → 須菩提之難 → 能生能示 → 母位確立」的脈絡；龍樹釋論八節剖判其辯證細節；末以中觀邏輯八節綜攝般若佛母三軌（能生、能示、能護），形成本卷對「般若佛母」命題的最完整論述。

承先啟後

前章已從「般若為諸佛母」的本體論層面確立般若母位；下一品〈大事起品〉接續問題：既已確立般若母位，為何菩薩能以一念心攝六波羅蜜？「大事」之大，何以為大？本品由現象（五事釋義）、譬喻（虛空喻）、實相（量不可得迴路）三層辯證，展開般若體用不二的中觀大事觀。

卷七十 · 第四品
大事起品
從現象到實相 · 般若體用不二
本品要旨

承前三品確立的「般若為諸佛母」「相不能知相」「般若深相」三軌，本品以「大事起」為題，由龍樹辯證階梯：事相 → 譬喻 → 實相，逐層展開。七章涵蓋導論、原典、五事釋義、四種法、量不可得迴路、虛空喻、法會果證與中觀總持，把卷七十四品的全部辯證收束在「大事起者，即般若體用不二之大事」的結論上。
